## Requirements
 - Ubuntu (or any debian-based distro)
 - make

## Installation

### Full

1. Clone this repo

```bash
git clone git@github.com:RamiGaggi/dotfiles.git ~/dotfiles
cd ~/dotfiles
```

2. Install

```bash
make all
```

### Partial

1. Clone this repo

```bash
git clone git@github.com:RamiGaggi/dotfiles.git ~/dotfiles
cd ~/dotfiles
```

2. Install

```bash
make packages-only
make fonts # kitty, oh-my-zsh, nvim, asdf
```
